Contents:

	"API" folder contains the main arduino API that is used to enable the device to recognise gestures made and send the output to a socket. 

	"demo" folder contains programs to received processed data from an arduino device with a neural network implemented to produce an output.

	"recording" folder contains programs to record data to train the neural network.
